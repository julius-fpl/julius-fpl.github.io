Über mich

Im Jahr 2019 lernte ich wie man Spiele mit der Unity Engine (C#) entwickelt. Es machte mir Spaß zu sehen, wie man mit wenigen Zeilen Code etwas erschaffen kann. Ich lernte dann wie man Windows Programme mit der Windows Presentation Foundation (WPF/C#) erstellen kann. Schnell habe ich herausgefunden, dass man mit dem Xamarine Framework (C#) Android Apps programmieren kann. Im Jahr 2022 lernte ich, wie man Android Apps mit der Android Studio IDE mit Java entwickeln kann und wie man eine Rest-API mit Python programmiert.
